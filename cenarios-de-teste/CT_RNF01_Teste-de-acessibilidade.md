## Requisito Não-Funcional 01: Acessibilidade da tela de perfil.

### Caso de Teste 01: Acessar e editar os campos de informações do usuário, apenas com teclado.

| ID         | Descrição                                                          |
| ---------- | ------------------------------------------------------------------ |
| RNF01-CT01 | Acessar aos todos os campos editáveis do perfil possibilitando o preenchimento sem uso de mouse. |

| **Pré-condições**                                                 |
| ----------------------------------------------------------------- |
| Usuário autenticado no sistema e utilizar somente o telado, mouse desconectado|

| **Passos**                                                        |
| ----------------------------------------------------------------- |
| **1-** Acessar a plataforma trindtech              |
|        carrega página inicial |
| **2-** Usando a tecla TAB acessar o 'Meu Perfil'   |
|        carrega página do perfil na aba Pessoal     |
| **3-** Navegar por todos os campos usando a tecla TAB |
|        ter acesso a botões e campos editáveis da tela |
| **4-** Verificar a ordem lógica do foco dos campos  |
|        percorrer todos os campos em ordem lógica   |

| **Resultados Esperados**                                        |
| ----------------------------------------------------------------- |
| Todos os campos estarem acessíveis utilizando apenas o teclado. 
  Indicador Visual de Foco sempre bem visível ao usuário.
  Seguir uma ordem lógica dos campos disponíveis na tela.   |
